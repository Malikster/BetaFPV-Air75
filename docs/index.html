Instructiuni de folosire

  • X / Hover este throttle-ul la care drona sta aproximativ in hover.
  • Expo Down controleaza finetea sub hover, adica la coborare.
  • Expo Up controleaza finetea imediat peste hover.
  • Up Scale reduce cat de mult influenteaza partea de sus a curbei.
  • Max throttle limiteaza throttle-ul maxim peste hover.
  • Daca vrei control mai fin imediat peste hover, cresti Expo Up.
  • Daca nu vrei ca throttle-ul sa urce pana la 100%, reduci Max throttle.
  • Daca vrei si mai putina putere peste hover, reduci Up Scale.
  • Linia portocalie/orizontala reprezinta Hover/X.
  • Linia verde/orizontala reprezinta Max throttle.
  • Tabelul arata valori utile pentru verificare in EdgeTX.

Recomandare initiala

X = valoarea reala de hover a dronei Expo Down = 0.65 Expo Up = 0.90 Up Scale = 0.60 Max throttle = 0.70
